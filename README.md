# Happy Father's Day 💙

A cute, card-style website that opens like a real envelope. Pure HTML/CSS/JS — no build step.

## View it

Double-click `index.html`, or run a tiny local server:

```bash
python -m http.server 8000
# then open http://localhost:8000
```

## Make it yours

- **Message** — edit the `<p class="message">` text in `index.html`.
- **Floating emojis** — change the `HEART_EMOJIS` array at the top of `script.js`.
- **Colors** — tweak the `:root` variables at the top of `styles.css`.

## Files

| File         | Purpose                                  |
|--------------|------------------------------------------|
| `index.html` | Structure: envelope, card, message       |
| `styles.css` | Look + the envelope-opening animation    |
| `script.js`  | Opening the card + floating hearts        |
